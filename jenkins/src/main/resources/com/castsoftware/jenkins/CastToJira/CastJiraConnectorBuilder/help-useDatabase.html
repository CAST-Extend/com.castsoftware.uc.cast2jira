Specifies the database server which contains the appliaction's action plan we want to transfer to Jira
  • CSS/Postgres: the selection of this option will setup the pluging to work with Cast Storage Service 1.0 & 2.0 and Postgres(RDS).
  • Oracle: the selection of this option will setup the pluging to work with Oracle 11gR2.
  • SQLServer: the selection of this option will setup the pluging to work with MS-SQLServer 2008.
